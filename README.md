# ButterDog Chat Ultra Deluxe

ButterDog Chat Ultra Deluxe is a web-based messaging application designed for fast, real-time communication. It consists of a Node.js backend and standard web technologies for the frontend (HTML, CSS, JS).

## Features

- User authentication with sessions
- Direct messaging and group chats
- Message editing and deletion
- Real-time chat functionality
- MongoDB integration for storing user data and chat history
